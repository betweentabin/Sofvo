## 確認手順

### 1. ログイン機能の確認

1. https://sofvo.pages.dev/#/login にアクセス
2. ログインフォームが表示されることを確認
3. メールアドレスとパスワードを入力
4. 「ログイン」ボタンをクリック
5. ホーム画面（`/home`）にリダイレクトされることを確認

### 2. ホーム画面の機能確認

#### 2.1 タブ切り替え
- 「フォロー中」タブと「おすすめ」タブが表示されることを確認
- タブを切り替えて、それぞれのコンテンツが表示されることを確認

#### 2.2 フォロー中タブ
- 「みんなの投稿」セクションが表示されることを確認
- 「いま投稿する」ボタンが機能することを確認
- 投稿カードが表示されることを確認（投稿がある場合）

#### 2.3 おすすめタブ
- 「大会を検索」機能が表示されることを確認
- 「最新の大会結果」セクションが表示されることを確認
- 「おすすめのプロフィール」セクションが表示されることを確認

### 3. 投稿機能の確認

#### 3.1 簡単投稿
1. 「いま投稿する」ボタンをクリック
2. 投稿フォームが表示されることを確認
3. テキストを入力
4. 画像をアップロード（任意）
5. 「投稿する」ボタンをクリック
6. 投稿がタイムラインに表示されることを確認

#### 3.2 大会結果投稿
1. フローティングボタン（右下の+ボタン）をクリック
2. 大会結果投稿フォームが表示されることを確認
3. 大会名、開催日時、開催地域、種別、試合結果、獲得ポイント、メモを入力
4. 「投稿する」ボタンをクリック
5. 投稿が表示されることを確認

### 4. 大会検索機能の確認

1. おすすめタブで「大会を検索」セクションを確認
2. 検索ボックスに大会名を入力
3. 「検索」ボタンをクリック
4. 検索結果画面（`/tournament-search-team`）に遷移することを確認

### 5. ナビゲーション機能の確認

#### 5.1 フッターナビゲーション
- ホーム、おすすめ、スケジュール、マイページ、メッセージのアイコンが表示されることを確認
- 各アイコンをクリックして、対応するページに遷移することを確認

#### 5.2 ヘッダーナビゲーション
- 設定アイコンが表示されることを確認
- 設定画面に遷移することを確認

### 6. マイページ機能の確認

1. フッターの「マイページ」アイコンをクリック
2. マイページ（`/my-profile`）に遷移することを確認
3. プロフィール情報が表示されることを確認
4. 「プロフィール編集」ボタンが機能することを確認

### 7. DM機能の確認

1. フッターの「メッセージ」アイコンをクリック
2. DM画面（`/dm`）に遷移することを確認
3. 会話リストが表示されることを確認（会話がある場合）

### 8. 設定画面の確認

1. ヘッダーの設定アイコンをクリック
2. 設定画面（`/settings`）に遷移することを確認
3. 各種設定項目が表示されることを確認：
   - 登録情報変更
   - 通知設定
   - 利用規約
   - プライバシーポリシー
   - お問い合わせ
   - 退会

### 9. 大会関連機能の確認

#### 9.1 大会スケジュール
1. フッターの「スケジュール」アイコンをクリック
2. 参加予定大会画面（`/tournament-schedule`）に遷移することを確認

#### 9.2 大会結果
1. ホーム画面の「最新の大会結果」から大会をクリック
2. 大会結果画面に遷移することを確認

### 10. API接続の確認

ブラウザの開発者ツール（F12）を開いて、以下を確認：

1. **Networkタブ**を開く
2. ページをリロード
3. APIリクエストが正常に送信されていることを確認
4. レスポンスステータスが200であることを確認
5. エラーが発生していないことを確認

### 11. レスポンシブデザインの確認

1. 開発者ツールでデバイスモードを有効化
2. モバイルサイズ（375px、414pxなど）で表示を確認
3. タブレットサイズ（768px、1024pxなど）で表示を確認
4. デスクトップサイズで表示を確認

## 自動テストスクリプトの使用方法

1. ブラウザの開発者ツール（F12）を開く
2. Consoleタブを開く
3. 以下のコードを貼り付けて実行：

```javascript
// テストスクリプトを読み込む
const script = document.createElement('script');
script.src = 'https://sofvo.pages.dev/test-browser-functionality.js';
document.head.appendChild(script);

// または、ローカルファイルから読み込む場合
// fetch('/test-browser-functionality.js')
//   .then(res => res.text())
//   .then(code => eval(code));
```

## 確認すべきエラー

以下のエラーが発生していないことを確認：

1. **コンソールエラー**
   - JavaScriptエラー
   - ネットワークエラー
   - APIエラー

2. **ネットワークエラー**
   - 404エラー（リソースが見つからない）
   - 500エラー（サーバーエラー）
   - CORSエラー

3. **認証エラー**
   - 401エラー（認証が必要）
   - 403エラー（権限がない）

## 確認結果の記録

各機能の確認結果を以下の形式で記録してください：

- ✅ 正常に動作
- ⚠️ 部分的に動作（問題あり）
- ❌ 動作しない

- APIサーバーが起動していることを確認してください
- データベースにテストデータが存在することを確認してください
